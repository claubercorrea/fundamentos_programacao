🚀 Fundamentos de Programação
Este repositório é o meu espaço dedicado à prática e ao estudo dos fundamentos de programação. Aqui, organizo os exercícios que desenvolvo para solidificar conhecimentos em diversas linguagens e tecnologias.

📂 Estrutura do Repositório
Cada pasta contém desafios, algoritmos ou projetos práticos correspondentes a uma linguagem ou tecnologia específica:

c#: Exercícios focados na linguagem C# e lógica de programação.

flask: Estudos e pequenos projetos utilizando o framework web Flask (Python).

javascript: Práticas de manipulação de DOM, lógica e conceitos essenciais de JS.

python: Scripts e exercícios variados para reforçar a sintaxe e a lógica em Python.

🛠️ Tecnologias Utilizadas
Este repositório utiliza uma combinação de linguagens que fazem parte da minha jornada de aprendizado:

C#

Python

HTML/CSS

JavaScript

🎯 Objetivo
O intuito deste repositório é documentar a minha evolução técnica, mantendo um histórico do que venho aprendendo e criando um portfólio de estudos constante.

"A prática constante é a base para o domínio de qualquer tecnologia."
